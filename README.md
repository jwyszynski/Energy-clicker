# ENERGY CLICKER

Prosta gra napisana w JAVASCRIPT HTML i CSS

## DEMO:

Wypróbować można na mojej stronie:  [http://info.wsisiz.edu.pl/~wyszynsj/Energy-clicker/](http://info.wsisiz.edu.pl/~wyszynsj/Energy-clicker/)
